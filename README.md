# mantan
htmlの制作課題
-必ず個人開発用のローカルブランチを切ってから編集すること
-編集するときはカレントブランチを確認すること
-コミットはこまめに、マージは完成してから


# clone後に実行するコマンド
$ cd mantan & ./setup.sh


# コマンド
-起動
$ make up
-終了
$ make down
